# ROM Archaeology

This directory contains ROM analysis work.

ROM annotations must follow `docs/rom-annotation-conventions.md`.
Do not add symbolic names, routine purposes, or behavior claims without either
evidence or an explicit hypothesis label.

Raw ROM images and reconstructive ROM-derived artifacts must not be stored in
this repository. Keep them under
`/home/tron/calc-archaeology-vault/private-rom/` and record only approved,
non-reconstructive metadata here. See
`skills/analyze-vintage-calculator-systems/references/rom-handling.md`.
